# 特别注意:

## 1: 很多用户反馈更新1.40后导致ALT+左键失灵, 原因是开启"妙控拖拽"功能后,使得按住ALT键会屏蔽左键与中键原功能!与原神等需要ALT点击界面的操作冲突! 请自行根据情况开关妙控拖拽功能. ALT+TAB可以用右键来选择哦!

2.对于现在少部分无法冻结的游戏, 再往深入搞就上升到内核对抗级别了,多见于有高级反调试的网游,比如有小蓝熊的APEX,老头环.![image](https://github.com/superDMS/HsFreezer-Hidden-in-the-snow-/assets/57837939/a56168dd-c107-40d4-8257-96bdc4c8a554)
驱动版本正在编写中,敬请期待.不过,目前也解决大量热门网游,包含但不仅限于:LOL,永劫无间,无畏契约,原神,崩铁,崩坏3(米哈游全攻略了)cs:go,我的世界等等等等(保证不触发反作弊等,因为无修改数据操作,这个有大量玩家已实测,可放心食用.)

3.使用各种查看帧数功耗OSD软件( 游戏加加,RTSS 等)的用户,请更新至1.13以上版本! 
此类软件属于强制内覆盖绘制软件,已经适配.

4.卡普空部分游戏:如生化危机等,尽量画面设置中尽量使用DX11,因为AMD驱动与DX12有某些冲突
(当然,这个驱动版本是玄学,也不一定需要这样做,可自行测试,现阶段,DX11更为稳定).

5.多设备玩steam游戏,由于云存档机制问题, 要注意有没有自动云存档，雪藏中的游戏,没退出前steam可能会没有把游戏存档即时上传更新, 换设备后, 会造成存档回滚. (感谢:古赫分享)
(单设备就尽情的造,设备太多也是一种烦恼哈哈)

6.经常休眠\睡眠唤醒后的用户,可以在后台长期冻住一个几十m的小游戏,每次休眠解除后,用来当解冻热身,也许会触发小概率的显存驱动崩溃,这样可保住主玩游戏,简称"舍车保帅"(能get到我在说啥么,工兵,帮踩雷)

7.有些朋友在某xxxx520下载的学习版单机, 冻结解冻后偶尔提示可能会提示致命错误, 这时候,你别点确定, 你照常玩确实没事,但你一点确定按钮就崩了,请悉知.

## 雪藏主界面>设置>使用说明栏:有各项功能的详解

 *加入sklearn高级加密算法,动态打包技术,按每个不同的机器环境,动态打包,因为py的静态打包机制,经常会出现放到另一个机子上无法运行的状况,所以首次部署都会从本机环境进行打包生成, 
 顺便调用sklearn的模板算法,从内存层面自动匹配各种不同的游戏引擎以及窗体特征(比特征码稳定,性能更好),做出最优的子窗体内覆盖判定,按最优的线程顺序逐步挂起,从底层提高稳定性.
    (本程序适侧重用于游戏,对其他进程无特别的适配,毕竟窗体匹配围绕着游戏引擎展开, 如需要的话,可自行对其他非游戏类进程进行测试.)


   (后台有朋友说: 这不就是资源JS器么? emmmm...看似同源,但细节差远了,对了,它使能比雪藏差不少,不信你冻崩坏3,冻无畏契约,冻LOL,试试. 一寸长一寸强,论冻结覆盖面与冻结能力,雪藏完胜.)
